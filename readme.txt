Add a DuckDuckGo search form to your site or blog to help people find what they need on your website.

== Description ==

[DuckDuckGo](https://duckduckgo.com/) is the search engine that doesn't track you. DuckDuckGo's mission statement is to protect internet searchers' privacy and avoid the filter bubble of personalized search results.

**DuckDuckGo Search** is a simple plugin that does one thing. It adds a search form in the widget area and it uses DuckDuckGo's search engine to search your site or blog.

Please note that this plugin isn't supported or maintained by DuckDuckGo. Any issues with this  plugin should be reported to the author and not to DuckDuckGo's support team.

To see this plugin in action go to https://www.hostingkingdom.com/blog/

== Installation ==

1. Upload the entire `duckduckgo search widget` folder to the to the `/wp-content/plugins/` directory - or just upload the ZIP package via 'Plugins > Add New > Upload' in your WP Admin area
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On the WordPress Widgets settings page simply drag `DuckDuckGo Search Widget` to your favorite widget area and you are done :)
